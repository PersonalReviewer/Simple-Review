# Changelog

## 0.1.0 - Initial public release

First public version of Review Studio.

### Added

- Cross-platform PySide6 desktop app
- Split-screen structured editor and live preview
- BBCode table output template
- Template-driven editor generation from JSON
- Standard rating guidance scale
- Stealth-specific rating guidance for Stealth Quality
- Local JSON review library
- Autosave
- Search
- Duplicate/delete review workflow
- Export to BBCode, Markdown, HTML, plain text, and JSON
- Dark/light theme setting
- Atomic JSON writes and `.bak` backup support
- Startup smoke check
- Unit and GUI smoke tests
- PyInstaller packaging scaffold

### Notes

- This is source-first. Packaged release builds still need platform testing.
- Screenshots are not included yet.
